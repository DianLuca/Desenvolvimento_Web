O que é CSS Inline?

O CSS Inline é uma forma de aplicar estilos diretamente aos elementos HTML usando o atributo style em cada tag.

Sobre o Site

Este site foi feito para demonstrar o uso de CSS Inline com foco em regras de cor, fonte e tamanho da fonte.
